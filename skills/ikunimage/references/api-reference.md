## Google 官方 7 条提示词技巧

来源：[Google 官方博客](https://blog.google/products-and-platforms/products/gemini/prompting-tips-nano-banana-pro)

### 技巧 1：使用具体细节描述

将主体、构图、动作、地点、风格都写清楚。

**提示词示例**：
```
一位亚洲年轻女性，精致五官，长发微卷，身穿白色连衣裙，
站在樱花树下，花瓣飘落，柔和的自然侧光，浅景深，背景虚化，
温暖色调，电影级质感，超高清细节，皮肤细腻通透
```

### 技巧 2：文字渲染能力

NanoBananaPro 支持在图片中渲染文字。

**提示词示例**：
```
一块复古风格的路牌，上面写着"Coffee Time"，
木质纹理背景，温暖灯光，街角咖啡店风格
```

### 技巧 3：信息图表 / 说明图

生成信息图表、教程图解。

**提示词示例**：
```
一张信息图表，展示如何制作珍珠奶茶，
步骤清晰，配图精美，配色采用柔和的粉色和白色
```

### 技巧 4：图像文字翻译

将图片中的文字翻译成其他语言。

**提示词示例**：
```
把这张图片中的英文菜单翻译成中文，
保持原有的排版和设计风格不变
```

### 技巧 5：风格 / 时间 / 焦点修改

可以通过提示词修改现有图片的风格、时间、焦点。

**提示词示例**：
```
将这个场景转换为夜景，聚焦在花朵上
```

### 技巧 6：宽高比调整与平台适配

根据不同平台调整图片比例：

| 平台 | 推荐比例 |
|------|----------|
| Instagram 帖子 | 1:1 或 4:5 |
| Instagram 故事 | 9:16 |
| YouTube 缩略图 | 16:9 |
| 小红书 | 4:5 或 1:1 |
| 抖音/短视频 | 9:16 |

### 技巧 7：多图融合 / 风格迁移 / 品牌一致性

将多张图片融合，保持品牌风格一致。

**提示词示例**：
```
将这两张图片融合为一张电影感的16:9图片，
并将人物的衣服换成第二张图片中的款式
```

## 提示词结构公式

### 六要素结构（推荐）

```
主体 + 动作 + 环境 + 艺术风格 + 光线 + 细节
```

### 详细提示词模板（中国风格）

```
[主体描述]：一位中国年轻女性，东方精致五官，黑色长发披肩
[服饰]：身穿新中式白色立领衬衫，搭配水墨印花长裙
[动作/姿态]：坐在中式茶室窗边，手持青瓷茶杯，轻微侧身
[环境/背景]：中式茶室内，木质格栅窗，竹帘半卷，柔和的暖色调灯光，浅景深
[艺术风格]：中国文艺摄影风格，电影级质感，东方含蓄美学
[光线]：自然侧光从花窗射入，营造温润雅致的氛围
[色调]：月白、黛绿、琥珀色为主调
[细节]：皮肤细腻，超高清，细节丰富
[约束]：无水印，无多余文字，画面干净，构图完整
```

## 错误码说明

| 状态码 | 含义 | 处理方式 |
|--------|------|----------|
| 200 | 成功 | 从 candidates[0].content.parts[0].inlineData.data 提取 base64 图片 |
| 400 | 请求错误 | 检查提示词或参数 |
| 401 | 未授权 | 检查 API Key |
| 429 | 频率限制 | 等待后重试 |
| 500 | 服务器错误 | 稍后重试 |
